Introduction

Base URL: http://localhost:8000 
This documentation aims to provide all the information you need to work with our API.

<aside>As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile).
You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).</aside>

Authenticating requests

This API is not authenticated.

curl --request GET \

POST api/auth/register

Example request:
curl --request POST \
    "http://localhost:8000/api/auth/register" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "firstname=b"\
    --form "lastname=n"\
    --form "sex=g"\
    --form "address=z"\
    --form "phone=m"\
    --form "email=gulgowski.asia@example.com"\
    --form "password=vYgxwmi/#"\
    --form "city=m"\
    --form "country=y"\
    --form "role=vendeur"\
    --form "avatar=@C:\Users\Papa Balla SOW\AppData\Local\Temp\php238B.tmp" \
    --form "id_card_recto=@C:\Users\Papa Balla SOW\AppData\Local\Temp\php239C.tmp" \
    --form "id_card_verso=@C:\Users\Papa Balla SOW\AppData\Local\Temp\php239D.tmp" 
const url = new URL(
    "http://localhost:8000/api/auth/register"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('firstname', 'b');
body.append('lastname', 'n');
body.append('sex', 'g');
body.append('address', 'z');
body.append('phone', 'm');
body.append('email', 'gulgowski.asia@example.com');
body.append('password', 'vYgxwmi/#');
body.append('city', 'm');
body.append('country', 'y');
body.append('role', 'vendeur');
body.append('avatar', document.querySelector('input[name="avatar"]').files[0]);
body.append('id_card_recto', document.querySelector('input[name="id_card_recto"]').files[0]);
body.append('id_card_verso', document.querySelector('input[name="id_card_verso"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

POST api/auth/register

Headers

Content-Type        

Example: multipart/form-data

Accept        

Example: application/json

Body Parameters

firstname   string     

Must not be greater than 255 characters. Example: b

lastname   string  optional    

Must not be greater than 255 characters. Example: n

avatar   file  optional    

Must be an image. Must not be greater than 2048 kilobytes. Example: C:\Users\Papa Balla SOW\AppData\Local\Temp\php238B.tmp

sex   string  optional    

Must not be greater than 255 characters. Example: g

address   string  optional    

Must not be greater than 255 characters. Example: z

phone   string     

Must not be greater than 255 characters. Example: m

email   string  optional    

Must be a valid email address. Must not be greater than 255 characters. Example: gulgowski.asia@example.com

password   string     

Must be at least 8 characters. Example: vYgxwmi/#

city   string  optional    

Must not be greater than 255 characters. Example: m

country   string  optional    

Must not be greater than 255 characters. Example: y

role   string  optional    

Example: vendeur

Must be one of:
  • acheteur
  • vendeur
id_card_recto   file     

Must be an image. Must not be greater than 2048 kilobytes. Example: C:\Users\Papa Balla SOW\AppData\Local\Temp\php239C.tmp

id_card_verso   file     

Must be an image. Must not be greater than 2048 kilobytes. Example: C:\Users\Papa Balla SOW\AppData\Local\Temp\php239D.tmp

Store a newly created resource in storage.

Example request:
curl --request POST \
    "http://localhost:8000/api/transactions/transfer" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"amount\": 16,
    \"wallet_uuid\": \"a4855dc5-0acb-33c3-b921-f4291f719ca0\",
    \"offline_id\": \"architecto\"
}"

const url = new URL(
    "http://localhost:8000/api/transactions/transfer"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "amount": 16,
    "wallet_uuid": "a4855dc5-0acb-33c3-b921-f4291f719ca0",
    "offline_id": "architecto"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Headers

Content-Type        

Example: application/json

Accept        

Example: application/json

Body Parameters

Must be at least 1. Example: 16

offline_id   string  optional    

Example: architecto
